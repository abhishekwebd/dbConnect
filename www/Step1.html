DBConnect is special type of android app which provide user to manage any type of database directly from their smartphone.
Currently only Sql and MySql Database can be accessed using DBConnect. User has to provide only server address, databse name, username and password to access database and he/she can perform all the operation on it.
Even there are options available to perform quick operation like :

  1. Select all tables from the database
  2. Select all procedures from the database
  3. View specific table data
  4. View specific procedure
  5. Delete single/all the tables from the database
  6. Delete single/all the procedures from the databse
  7. truncate single/all the tables from the database
